### What I ship

| | Project | What it is |
|---:|---|---|
| 🤖 | [**RoboSharp**](https://github.com/frankhaugen/RoboSharp) | Small C#-style language + **teaching compiler** — lexer → trees → binder → IL → robot on a grid. |
| 🎮 | [**Frank.GameEngine**](https://github.com/frankhaugen/Frank.GameEngine) | Game engine **decoupled from graphics** — simulation and rules first. |
| 🪟 | [**Frank.Wpf**](https://github.com/frankhaugen/Frank.Wpf) | **WPF** building blocks — hosting, markdown, controls, sane patterns. |
| ⏱️ | [**Frank.CronJobs**](https://github.com/frankhaugen/Frank.CronJobs) | Cron-style scheduling built on **CronQuery** ideas — experimental extensions. |
| 📡 | [**Frank.Channels.DependencyInjection**](https://github.com/frankhaugen/Frank.Channels.DependencyInjection) | **`Channel<T>`** as a first-class DI resource — small, sharp API. |
| 🪨 | [**Frank.BedrockSlim**](https://github.com/frankhaugen/Frank.BedrockSlim) | **Bedrock**-style LLM client/server split — NuGet `Server` / `Client`. |
| 📈 | [**Frank.PulseFlow**](https://github.com/frankhaugen/Frank.PulseFlow) | **In-process messaging** on `Channel<T>` — Nexus, Conduit, Pulse, Flow. |
